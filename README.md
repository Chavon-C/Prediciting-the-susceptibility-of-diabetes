# Prediciting-the-susceptibility-of-diabetes
This project develops a logistic regression model to predict diabetes diagnosis among 532 Akimel O’odham women aged over 21 in Southern Arizona. Using health indicators including glucose concentration, BMI, blood pressure, skin fold thickness, number of pregnancies, age, and family history, I estimated and compared multiple logit specifications to identify the model with the strongest classification performance (using metrics such as AUC and accuracy). I then interpreted coefficients and marginal effects to quantify how much each characteristic contributes to diabetes risk, and translated the results into targeted public health recommendations to help a healthcare trust prioritise screening and intervention efforts for high-risk women.
